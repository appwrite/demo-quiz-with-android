# 🤖 Demo Quiz with Android

Simple quiz application built with Android and Appwrite

## 🚀 Usage

- Clone the repository

```sh
git clone https://github.com/lohanidamodar/demo-quiz-with-android
```

- Import into Android Studio and run!
